<CardGrid stagger>
  <Card title="Multi-Agent Pipeline" icon="setting">
    Coordinates PM, designer, engineer, reviewer, and tester agents through a
    declarative, config-driven pipeline with self-correcting review loops.
  </Card>
  <Card title="Parallel Execution" icon="rocket">
    Independent tasks run simultaneously in separate streams. Wave-based
    execution handles dependencies automatically.
  </Card>
  <Card title="Cross-Model Review" icon="approve-check-circle">
    A different AI model reviews the final output, catching systematic blind
    spots and biases specific to the primary model.
  </Card>
  <Card title="Fleet Mode" icon="globe">
    Orchestrate features across multiple repositories with shared contracts,
    dependency-ordered execution waves, and cross-repo consistency review.
  </Card>
